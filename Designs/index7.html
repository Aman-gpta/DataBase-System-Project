Info

This project addresses a critical need within educational institutions by introducing a user-friendly and intuitive database system that streamlines the process of scheduling and managing extra classes for teachers. Furthermore, the system will provide a platform for teachers to report and track classroom issues promptly, ensuring a conducive learning environment for students.To build a reliable and efficient system, this project requires proficient knowledge of database administration and programming languages such as SQL.

Objective

The scope of this project involves designing and implementing a comprehensive database management system tailored for classroom management. The system aims to facilitate teachers in scheduling and conducting extra classes efficiently. Additionally, it will enable reporting of classroom issues such as malfunctioning lights, AC, fans, projectors, and any other relevant equipment.

Project Description

The project involves creating a database schema, using SQL to run the database, creating an interface for the user, thoroughly testing the system, putting it into production, and making sure it is maintained continuously. Class Scheduling Module: Allows teachers to schedule extra classes based on their availability and student needs.

Teacher Availability Tracking: Enables teachers to indicate their availability for conducting extra classes, facilitating efficient resource allocation.
Issue Reporting: Allows teachers to report any classroom issues encountered during extra classes, including malfunctioning lights, AC, fans, projectors, or other equipment.
